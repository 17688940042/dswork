市旅游质监所党支部 到莲麻村开展主题党日活

3月23日，市旅游质监所党支部及共建单位、市中级人民法院民事庭组织党员到从化莲麻村开展主题党日活动，送法到基层、服务到一线，达到“群众得实惠、党员受教育”的效果。
广州市全域旅游创建工作现场推进会在番禺区召开

为进一步推进我市全域旅游发展，加强全域旅游示范区创建工作经验交流，市旅游局于3月15日在番禺区组织召开广州市全域旅游创建工作现场推进会。
